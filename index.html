BoninPlayer

BoninPlayer est un logiciel android ou windows qui permet de lire des fichier MP3 et MP4. Pour le télécharger choisissez votre système.

Cliquez sur le bouton ci-dessous pour télécharger l'application BoninPlayer sur votre appareil Android.

Télécharger pour Android Télécharger pour Windows

Quand vous-ètes sur le site github.com en cliquand sur Télécharger pour Windows, cliquer sur "BoninPlayer.exe".

Assurez-vous d'autoriser l'installation d'applications depuis des sources inconnues.
